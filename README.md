## Data y comentarios sobre el desafío

### Rasgos generales

Está creado con create-react-app, pensé en usar bootstrap pero creí conveniente hacer mis propios estilos para ser más auténtico.
La idea principal es hacerlo simple, funcional y agradable a la vista, tanto el código cómo el resultado final.
Voy a utilizar Redux, Router, Sass y jsx para los componentes.

### Estructura

La organización del repositorio será bastante convencional, dentro de la carpeta src estará en index.js junto a tres carpetas:
  -pages: donde se encontrará el archivo App.js (cumpliendo la función de layout)
  -components: ahí podrán encontrarse los tres fundamentales, home, login y navdrawer.
  -assets: estará destinada a estilos, tendrá archivos y carpetas contenedoras de mixins y variables.

### Tiempo estimado

Actualmente me encuentro bastante atareado laboralmente pero espero poder finalizarlo durante el fin de semana.
